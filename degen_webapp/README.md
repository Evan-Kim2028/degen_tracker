# Degen Tracker Webapp

## Getting started

Install [node](https://nodejs.org/en/download) via download or a manager such as brew

```bash
  npm install
```

## Development

```bash
  npm run dev
```

## Build

```bash
npm run build
npm run start
```

Runs on http://localhost:3000 by default.
